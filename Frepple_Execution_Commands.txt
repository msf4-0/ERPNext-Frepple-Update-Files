## Execute the following commands one-by-one after you login into "ERPNext-Frepple-Update-Files" folder > "cd ERPNext-Frepple-Update-Files" folder":

docker cp frepple\frepple\doctype <project_name>-erpnext-python-1:\home\frappe\frappe-bench\apps\frepple\frepple\frepple

docker cp frepple\config\frepple.py <project_name>-erpnext-python-1:\home\frappe\frappe-bench\apps\frepple\frepple\config


Notes:
- Replace <project_name> by your project name such as project1
- For example: docker cp frepple\frepple\doctype project1-erpnext-python-1:\home\frappe\frappe-bench\apps\frepple\frepple\frepple
- If you face "no such container" error, change "<project_name>-erpnext-python-1" with:
	* <project_name>-erpnext-python_1
	* <project_name>_erpnext-python-1
	* <project_name>_erpnext-python_1
